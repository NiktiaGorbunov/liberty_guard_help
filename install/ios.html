Инструкция по установке VPN

Следуя этой инструкции, вы легко сможете установить наше приложение на своё устройство. После подключения весь ваш интернет-трафик будет зашифрован, и вы сможете безопасно пользоваться сетью, даже при работе в публичных Wi-Fi сетях или в странах с ограничениями доступа.

  • Главная (FAQ)

1. Установка приложения

  1. Скачайте приложение v2RayTun из App Store

2. Получите ключ

  1. В боте получите ключ.

3. Настройка приложения

  1. Скопируйте вашу персональную ссылку для подключения.
  2. Запустите приложение v2RayTun и нажмите на кнопку "+" в правом верхнем углу.
  3. Выберите "Добавить из буфера". На открывшейся вкладке подключения нажмите "Сохранить"
  4. Внизу страницы приложения появится надпись "Subscription". Нажмите на значок "Обновить".
  5. Свайпните страницу приложения вверх. Нажмите на кнопку включения для активации защищенного соединения.

4. Возможные ошибки

  • Не поддерживается версия устройства или приложения.
  • Проблема с подпиской — проверьте срок действия.
