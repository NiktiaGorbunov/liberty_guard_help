3. Настройка приложения

  1. Скопируйте вашу персональную ссылку для подключения.
  2. Запустите приложение v2RayTun и нажмите на кнопку "+" в правом верхнем углу.
  3. Выберите "Добавить из буфера". На открывшейся вкладке подключения нажмите "Сохранить"
  4. Внизу страницы приложения появится надпись "Subscription". Нажмите на значок "Обновить".
  5. Свайпните страницу приложения вверх. Нажмите на кнопку включения для активации защищенного соединения.

4. Возможные ошибки

  • Не поддерживается версия устройства или приложения.
